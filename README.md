**Repository for the manuscript submitted to *Nature Communications***  
**Title:** Fault-Tolerant Quantum Simulation of Multi-Decadal Resonant Oscillations in Automation-Driven Labour–Capital Substitution

## Overview

This repository contains all code, data, and scripts required to reproduce the results of the paper. We demonstrate the first fault-tolerant quantum simulation of resonant labour–capital substitution dynamics using surface-code logical qubits. The work shows that logical qubits maintain coherent multi-decadal oscillations while physical qubits suffer rapid decoherence, establishing a clear pathway toward quantum advantage in macroeconomic modelling.

## Key Results

- Fault-tolerant simulation of 54.1-year resonant oscillations using logical qubits protected by the surface code.
- Direct comparison of ideal classical, noisy physical-qubit, and fault-tolerant logical-qubit simulations (Figure 1).
- Quantum advantage scaling: logical qubits preserve high fidelity at system sizes where classical tensor-network and physical-qubit methods collapse (Figure 2).
- Resource estimates showing that a few hundred logical qubits suffice for realistic multi-sector macroeconomic forecasting.
